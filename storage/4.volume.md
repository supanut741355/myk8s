
# Volumes

to persist data by container/pod -> attach volume to container

(data from container/pod will place in volume)


# Volumes and mounts

pod
```
apiVersion: v1
kind: Pod
metadata:
  name: ran-number-generator
spec:
  containers:
  - name: alpine
    image: alpine
    command: ['/bin/sh', '-c']
    args: ['shuf -i 0-100 -n 1 >> /opt/number.out']
    volumeMounts:
      - mountPath: /opt
        name: data-volume
  volumes:
  - name: data-volume
  hostPath:
    path: /data
    type: Directory
```


### create volume
```
spec:
  volumes:
  - name: data-volume
  hostPath:
    path: /data
    type: Directory
```

### mount the volume
```
spec:
  container:
  - name: yoyo
    volumeMounts:
    - mountPath: /opt
      name: data-volume
```

### Persist Volume
: manage storage centrally
eg.
admin creat a large pool of storage and take some piece to each pod.


user can select the PV for the pod/app by using Persist Volume Claims (PVC).


```
spec:
  accessModes:
  - [MODE]
  capacity:
    storage: [SIZE]Gi
  hostPath:
    path: /tmp/data
  [CLOUD_PROVIDOR_SERVICE]:
    volumeID: [VOLUME_ID]
    fsType: ext4
```

mode lists:
- ReadOnlyMany
- ReadWriteOnce
- ReadWriteMany

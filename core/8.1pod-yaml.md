# pod manifest


```
apiVersion: v1
kind: Pod
metadata:
  name: myapp-pod
  labels:
    app: myapp
    type: front-end

spec:
  containers:
    - name: nginx-container
      image: nginx
```


*labels can used for filtering


Create
```
$ kubectl apply -f <pod-def>.yml
```

Read
```
$ kubectl get pod
```

Read Info
```
$ kubectl describe pod <metadata.name>
```

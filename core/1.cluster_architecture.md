K8s cluster has 2 main component 
1. Master node / Control plane
2. Worker node

# Master node control plane
responsile for manage all operation of kube cluster

has 4 components
1. Kube API server -> manage all component
2. etcd -> kv db of cluster 
3. Controller manager
  3.1 Node controller
  3.2 ReplicaSet controller
  3.3 Job, Cronjob controller
4. Kube-scheduler -> manage and locate container to node



# Worker node
resposible for real workload

has 3 components
1. Kubelet -> manage deploy, edit or destroy container of node
2. Kube-proxy -> network, routing and communication between pod or node
3. Contrainer-runtime -> run workload container

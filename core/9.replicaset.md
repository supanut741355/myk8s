# ReplicaSet


If we have only one pod is risk to down time, so for HA we need to increase the number of pod. The thing that 
will handle the pod is **replicaSet**.


ReplicaSet responsibility is manage the number of pod = balance the load (Scale) and make it HA.



---
```
apiVersion: apps/v1
kind: ReplicaSet
metadata:


spec:
  template:
  ""
    pod-manifest
  ""
  replicas: 3
  selector:
    matchLabels:
      xxxxx: yyyyy

```

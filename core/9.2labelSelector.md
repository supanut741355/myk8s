# Label and Selector

is mechanism to bing main object to under object.

Q: Why do we label pod and other k8s' object ?
A: case we deploy 3 pod so how can ensure the 3 pod still alway active all the time -> Use the replicaset. The replica set will monitor and manage number of pod. So how the repicaSet know whcich pod is need to minitor -> using the **Label and Selector** to help this. 

matchLabel will filter the label, same label will bind the pod. -> this is the way to let replicaSet konw and monitor the pod.

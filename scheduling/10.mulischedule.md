# Mutli scheduler

In case u have own scheduling algorithm to place pod on node.

when create the pod or deployment -> can instruct k8s to have a pod scheulded by **specific schedule**


### Default scheduler
```
apiVersion: kubescheduler.config.k8s.io/v1
kind: KubeSchedulerConfiguration
profiles:
  - schedulerName: default-scheduler
```


### Own custom scheduler
can specific name

```
apiVersion: kubescheduler.config.k8s.io/v1
kind: KubeSchedulerConfiguration
profiles:
  - schedulerName: my-scheulder1
```

```
apiVersion: kubescheduler.config.k8s.io/v1
kind: KubeSchedulerConfiguration
profiles:
  - schedulerName: my-scheulder2
```

### Deploy an additional scheduler
1. download kube-scheduler binary
```
$ wget https://storage.gooleapi.com/kubernetes-relaese/relaease/v1.xx.0/bin/linux/amd64/kube-scheduler
```

2. Deploy a additional scheduler by fix file  kube-scheduler.service
```
ExecStart=/usr/local/bin/kube-scheduler --config=/etc/kubernetes/config/kube-scheduler.yaml
```

```
ExecStart=/usr/local/bin/kube-scheduler --config=/etc/kubernetes/config/my-scheduler-config.yaml
```

```
ExecStart=/usr/local/bin/kube-scheduler --config=/etc/kubernetes/config/my-scheduler-2-config.yaml
```

### Another way to deploy scheduler as a pod

<my-custom-scheduler.yaml>
```
apiVersion: v1
kind: Pod
metadata:
  name: my-custom-scheduler
  namespace: kube-system
spec:
  containers:
  - name: kube-scheduler
    image: k8s.gcr.io/kube-scheduler-amd64:v1.XX.X
    command:
    - kube-scheduler
    - --address=127.0.0.1
    - --kubeconfig=/etc/kubernetes/scheduler.conf
    - --config=/etc/kubernetes/my-scheduler-config.yaml
```

<my-scheudler-config.yaml>
apiVersion: kubescheduler.config.k8s.io/v1
kind: KubeSchedulerConfiguration
profiles:
  - schedulerName: my-custom-scheduler
leadElection:
  leaderElect: true
  resourceNamespace: kube-system
  resourceName: lock-object-my-scheduler


### Use custom scheduler
```
apiVersion: v1
kind: Pod
metadata:
  name: nginx
spec:
  containers:
    - name: nginx
      image: nginx
  sechedulerName: my-custom-scheduler
```

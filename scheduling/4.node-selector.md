# Node selector

normally pod can place on every node, but some pod replace at node that not desire => to fix it we will limit that run on specific node has 2 ways

1.nodeSelector
```
spec:
  containers:
  - name:
    image:
  nodeSelector:
    <KEY>:<VALUE>
```

KEY is lable that we attach on node 


### How to lable node

```
$ kubetl label node <NODE_NAME> <LABEL_KEY>=<LABEL_VALUE>

eg.

$ kubectl label node node01 size=large

```

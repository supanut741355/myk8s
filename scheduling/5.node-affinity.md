# Node affinity
for advance expression to **limit pod placement on specific node**

```
affinity:
  nodeAffinity:
    requireDuringSchedulingIgnoredDuringExecution:
      nodeSelectorTerms:
      - matchExpressions:
        - key: size
          operator: In 
          values:
          - large
          - medium
```


or invert expression
```
          operator: NotIn
          values:
          - small
```


# Node Affinity type 
3 types:
<Available>
1. requireDuringSchedulingIgnoreDuringExcution
2. preferredDuringSchedulingIgnoreDuringExcution
<plan>
3. requireDuringSchedulingRequireDuringExcution


### requireDuringSchedulingIgnoreDuringExcution
Require During Scheduling: ต้องเป็นไปตามเงื่อนไขตอนเลือก node
Ignore  During Excution: แต่พอรันไปแล้ว จะไม่สนใจเงื่อนไขนั้นอีก 
(ข้อบังคับตอน schedule)

behavior:
- ถ้า node ไม่ตรงเงื่อนไข → Pod จะไม่ถูก schedule เลย
- ถ้ามีการเปลี่ยนแปลง label ของ node ทีหลัง → Pod จะยังรันต่อ (ไม่โดน evict)

eg
```
affinity:
  nodeAffinity:
    requiredDuringSchedulingIgnoredDuringExecution:
      nodeSelectorTerms:
        - matchExpressions:
            - key: disktype
              operator: In
              values:
                - ssd
```
Pod นี้ “ต้อง” ลง node ที่มี disktype=ssd เท่านั้น

---

### preferredDuringSchedulingIgnoreDuringExcution

preferred During Scheduling:  เป็นแค่ความต้องการ (ไม่บังคับ)

Ignore    During Excution: ตอนรันแล้วก็ไม่สนเหมือนกัน
(เป็น “ความต้องการ (soft rule)” ให้ scheduler พยายามเลือก)

behaviors:
- ถ้ามี node ที่ตรงเงื่อนไข → จะเลือก node นั้นก่อน
- ถ้าไม่มีเลย → ก็ยัง schedule ได้ (ไม่ fail)
- มีระบบ weight (1–100) เพื่อบอกว่าชอบแค่ไหน

eg.
```
affinity:
  nodeAffinity:
    preferredDuringSchedulingIgnoredDuringExecution:
      - weight: 10
        preference:
          matchExpressions:
            - key: disktype
              operator: In
              values:
                - ssd
```
“อยาก” ลง node ที่เป็น ssd แต่ถ้าไม่มี ก็ไป node อื่นได้


---

required... = “ต้องได้แบบนี้เท่านั้น”
preferred... = “ถ้าได้แบบนี้จะดีมาก แต่ไม่มีก็ไม่เป็นไร”
